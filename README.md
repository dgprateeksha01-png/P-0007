# P-0007
MEDICINE REMINDER APP 
💊 Medicine Reminder App
📌 Project Description

The Medicine Reminder App is a web/mobile application designed to help users take their medicines on time. It allows users to add medicines, set reminders, and track their medication history. The app sends notifications to ensure users never miss a dose, improving health management.

🚨 Problem Statement

Many people forget to take medicines on time, especially elderly patients and busy individuals. Missing doses can lead to serious health issues. This application solves that problem by providing an easy and reliable reminder system.
